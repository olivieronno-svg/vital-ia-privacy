# Suppression de vos données — VITAL I.A

**Dernière mise à jour : 13/05/2026** · Application : `fr.ambu.bilan`

VITAL I.A traite la **quasi-totalité de vos données localement sur votre appareil**. Aucun compte utilisateur n'est créé sur nos serveurs et aucune fiche bilan n'est envoyée à l'éditeur.

---

## Données stockées sur votre appareil

| Type de données | Stockage |
|---|---|
| Fiches bilan patient (identité, examen, gestes, destination…) | Base Room locale |
| Calendrier de formations (AFGSU, PSE, permis…) | Base Room locale |
| Préférences (langue, profil pro/famille, bouteille O₂ par défaut, son d'introduction…) | DataStore local |
| Badges quiz et XP cumulés | DataStore local |

**Aucune de ces données ne quitte votre téléphone.**

---

## Comment supprimer toutes vos données ?

### Option 1 — Effacer les données de l'application (recommandé)

1. Ouvrez les **Paramètres** de votre téléphone Android.
2. Allez dans **Applications** → **VITAL I.A**.
3. Choisissez **Stockage et cache** → **Effacer les données** (puis confirmez).

Cela supprime instantanément toutes les fiches bilan, le calendrier, les badges, les XP et les préférences.

### Option 2 — Désinstaller l'application

Désinstaller VITAL I.A depuis l'écran d'accueil ou le Play Store supprime automatiquement toutes les données locales associées.

## Données gérées par Google (achats Premium)

L'abonnement Premium est géré par **Google Play Billing**. Pour le résilier ou pour toute question sur l'historique d'achat :

- Ouvrez **Google Play Store** → photo de profil → **Paiements et abonnements** → **Abonnements**.
- Ou contactez directement le **support Google Play** : <https://support.google.com/googleplay>.

L'éditeur Onn-Off n'a pas accès aux données de facturation Google Play.

---

## Données envoyées à des services tiers

VITAL I.A **n'envoie aucune donnée patient** à des serveurs distants. Les seules interactions externes sont :

- **Google Play Billing** : uniquement pour les achats Premium (géré par Google).
- **Reconnaissance vocale Android** : la dictée (saisie vocale du bilan) utilise le service Android de reconnaissance vocale, qui peut transmettre votre voix à Google selon les paramètres système de votre appareil. Vous pouvez désactiver cette fonctionnalité en n'utilisant pas le bouton micro.
- **OpenStreetMap (API Overpass)** : pour rechercher les défibrillateurs (DAE) et les hôpitaux à proximité. **Seules vos coordonnées GPS approximatives sont envoyées**, sans identifiant.

Aucun service d'analyse comportementale, aucun cookie, aucun pixel de tracking.
